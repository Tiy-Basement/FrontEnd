# [Basement](https://github.com/Tiy-Basement/FrontEnd)

Basement is a web-based application to schedule group events based around individuals' schedules.


### Libraries, frameworks, sub-modules, etc.

- Foundation 
- Angular
- Angular UI-Calendar
- FullCalendar
- jQuery
- Moment
